🚀 PRÓXIMOS PASOS - INSTRUCCIONES RÁPIDAS
════════════════════════════════════════════════════════════════════════════

1️⃣  INSTALAR DEPENDENCIAS (5 minutos)

    cd C:\Projects\Agents3B
    npm install

2️⃣  CONFIGURAR VARIABLES DE ENTORNO (10 minutos)

    cp .env.example .env
    # Editar .env con tus valores:
    # - ANTHROPIC_API_KEY (de https://console.anthropic.com)
    # - AZURE_ACCESS_TOKEN o credenciales Service Principal
    # - SENDGRID_API_KEY (de https://sendgrid.com)

3️⃣  VERIFICAR INSTALACIÓN (2 minutos)

    npm run build
    # Debe completarse sin errores

4️⃣  PROBAR EN MODO CHAT (5 minutos)

    npm run chat
    # Escribe algo y presiona Enter
    # Escribe /exit para salir

5️⃣  LEER DOCUMENTACIÓN (20 minutos)

    Orden recomendado:
    1. README.md               → Uso general
    2. CLAUDE.md              → Contexto técnico
    3. SETUP_QUICK_START.md   → Detalles de setup
    4. PROJECT_CONTEXT.md     → Referencia completa
